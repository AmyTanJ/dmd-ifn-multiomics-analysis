# scRNA-seq and RNA Velocity Analysis

This directory contains workflows for **single-cell RNA-seq (scRNA-seq) analysis** and RNA velocity inference in Duchenne muscular dystrophy (DMD) studies.

---

## Contents

### scRNA-seq Analysis
- Cell filtering, normalization, and clustering
- Visualization of gene expression and cell states
- Integration and comparison across datasets

### RNA Velocity
- Preprocessing using Velocyto
- Velocity inference and visualization using scVelo
- Investigation of transcriptional dynamics and state transitions

### CellSexID Prediction
- Sex prediction outputs generated using CellSexID
